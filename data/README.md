# Carpeta de datos

Coloca aquí los archivos de datos antes de ejecutar el script:

## Datos en bruto (raw)
- `bank-additional.csv` — Dataset principal de campañas bancarias (43.000 registros)
- `customer-details.xlsx` — Dataset secundario con detalles de clientes (20.115 registros)

## Datos transformados (generados automáticamente)
- `bank_clean.csv` — Dataset unificado y limpio generado al ejecutar `src/eda_banking.py`

> Los archivos de datos no están incluidos en el repositorio por su tamaño.
> Puedes obtenerlos en la plataforma del máster.
